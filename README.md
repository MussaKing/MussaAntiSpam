# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/MussaKing/MussaAntiSpam.git
cd MussaAntiSpam
```

## Install Dependencies

### Node.js

```bash
npm install
```

### Python

```bash
pip install -r requirements.txt
```

---

# ⚙️ Configuration

Create `.env` file:

```env
BOT_TOKEN=your_bot_token
OWNER_ID=your_telegram_id
```

---

# 🚀 Run Bot

### Node.js

```bash
npm start
```

### Python

```bash
python main.py
```

---

# 📋 Commands

| Command | Description |
|---|---|
| `/start` | Start bot |
| `/help` | Show help menu |
| `/ban` | Ban user |
| `/mute` | Mute user |
| `/unmute` | Unmute user |
| `/warn` | Warn user |
| `/antispam on` | Enable anti-spam |
| `/antispam off` | Disable anti-spam |
